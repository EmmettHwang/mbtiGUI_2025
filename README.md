### 설명 

### 1. 🛠️ 콘솔 MBTI검사에 비해서 주요 변경 사항 및 특징
  tkinter 사용: 파이썬 설치 시 기본으로 포함된 라이브러리를 사용하여 추가 설치(pip install)가 필요 없습니다.

  클래스(class) 구조: MBTIGUI라는 클래스 안에 기능들을 묶어 관리하므로 코드가 체계적이고 확장이 쉽습니다.

  시각적 요소:

  진행 상황 표시: 상단에 진행 상황: 1 / 12와 같이 현재 위치를 보여줍니다.

  버튼 디자인: A안과 B안을 쉽게 구분할 수 있도록 배경색(bg)과 크기(width, height)를 조정했습니다.

  결과 화면: 모든 질문이 끝나면 질문 버튼들이 사라지고, 결과 유형과 해설을 큰 글씨로 보여줍니다.

### 2. ▶️ 실행 방법

  메모장이나 파이썬 에디터(VS Code, PyCharm 등)에 붙여넣습니다.

  mbti_gui.py라는 이름으로 저장합니다.

  저장된 파일을 실행하면 윈도우 창이 뜹니다.
